## Краткое описание работы
Цель работы:
- Познакомиться с языком программирования JavaScript.
- Понять его место в web-разработке.
- Сравнить базовые концепции JavaScript и C#.
- Научиться запускать JavaScript-код в браузере и через Node.js.

## Описание (что изучили)
В ходе выполнения лабораторной работы были освоены следующие темы:
1. **Подготовка окружения:** Настройка VS Code, инициализация Git-репозитория, публикация проекта на GitHub.
2. **Подключение JS к HTML:** Использование тега `<script>`, атрибута `defer`, порядок загрузки скриптов.
3. **Переменные и типы данных:** Объявление через `let`, `const`, `var`. Динамическая типизация, примитивные и объектные типы.
4. **Проверка типов:** Использование оператора `typeof`, особенности проверки `null`.
5. **Преобразование типов:** Явное (`Number()`, `String()`) и неявное преобразования, правила конкатенации и арифметики.
6. **Сравнение:** Строгое (`===`) и нестрогое (`==`) сравнение, сравнение объектов по ссылке.
7. **Работа с консолью:** Использование `console.log()`, отладка в браузере.
8. **Node.js:** Установка, запуск скриптов вне браузера, сравнение с .NET Runtime.
9. **Условные операторы:** `if/else`, `switch/case`, тернарный оператор, логические операторы.
